• SMS
    SAT Modulo Symmetries (SMS)

    A SAT-based framework for exhaustive graph-theoretic search — supports graphs, digraphs, planar graphs, matroids, and other combinatorial structures by exploiting symmetries to prune the search space.

    Further Contributors Markus Kirchweger, Stefan Szeider
    Software docs · GitHub
    Read SAT-Based Generation of Planar Graphs (SAT 2023)
  • RotSys
    RotSys-SAT

    A SAT-based framework for investigating topological drawings via rotation systems — enables automated search and verification of combinatorial properties of graph drawings.

    Further Contributors Helena Bergold
    Software GitHub
    Read Investigating Simple Drawings of Kn using SAT (arXiv:2504.02650)
  • Pseudocircle Arrangements

    Tools and an extensive data collection for arrangements of pseudocircles, covering enumeration, realizability, and structural properties.

    Further Contributors Stefan Felsner
    Data & Software data website · GitHub
    Read Triangles and Drawings (DCG 2021) · On Circularizability (DCG 2020) · On Digons and Triangles (CGT 2024) · Coloring Circle Arrangements (EJC 2023) · Flip Graph Connectivity (SODA 2024)
    Also used in The Unavoidable Arrangements of Pseudocircles (AMS 2019) — "experimenting with this code was very useful at the early stages of this project"
  • kgon
    Contact Representations (k-gon)

    Algorithms and theory for representing graphs as contact graphs of convex polygons; includes a full computational study of k-gon contact representations.

    Further Contributors Stefan Felsner, Hendrik Schrezenmaier
    Data & Software project page
    Read dissertation (Schrezenmaier, TU Berlin 2023)
  • Point Set Zoo (PoSeZo)

    A curated collection of point sets with rare combinatorial properties — both interesting individual configurations and well-known families from the literature (e.g. Horton sets) — serving as a reference resource for researchers in combinatorial and computational geometry.

    Further Contributors International collaboration of many experts · members
    Data website
